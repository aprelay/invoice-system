## Best Practices

### 1. Rotate Templates Regularly
- Don't use the same template repeatedly
- Mix original (1-7) and new (8-29) templates
- Change templates every 5-10 emails

### 2. Match Template to Urgency
- **Urgent**: Template 16 (Simple Red), Template 25 (Rich Burgundy)
- **Normal**: Templates 8-15 (cool colors)
- **Friendly**: Template 20 (Soft Pink), Template 27 (Vibrant Coral)

### 3. Brand Alignment
- Choose template colors that match your brand
- Use consistent template for same customer
- Create template preference per client

### 4. Monitor Performance
- Track which templates have best inbox rate
- Note customer responses per template
- Adjust strategy based on data

## Troubleshooting

### If Emails Go to Spam
1. Try templates 8-29 (higher deliverability)
2. Use generic button texts
3. Avoid templates 1-7 temporarily
4. Check SPF/DKIM/DMARC records

### If Templates Look Wrong
1. Test in Office365 Outlook first
2. Check mobile email client rendering
3. Verify colors display correctly
4. Ensure button text is readable
